# Job-market-analyzer
This project analyzes job market data using Python ans CSV files.
Features: 
-  Show total number of jobs
-  Calculate average salary
-  Find highest salary job
-  Search jobs by location
-  Seacrh jobs by level
-  Search jobs by skills
-  Count job titles
-  Display all jobs

Technology used:
- Python
- Excel
